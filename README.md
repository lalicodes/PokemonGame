## Prerequisites
C Compiler (e.g., gcc)
Make (build tool)

## Installation
Clone the repo and build the game using the provided Makefile.

## Clone the repo
git clone git@github.com:lalicodes/PokemonGame.git

## Navigate to the project directory
cd PokemonGame

## Build and Run
Build the game:
make

## Run the game:

make run
Clean the build files:
make clean
